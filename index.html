Typography Scale CSS

A scalable and customizable typography system for your projects.

Why Use This CSS?

This CSS system provides a structured way to scale typography dynamically across different screen sizes using CSS variables. It ensures readability and consistency in your UI design.

Implementation Guide

  1. Download the CSS file using the button below.
  2. Link it in your HTML file: <link rel="stylesheet" href="style.css">
  3. Apply the typography classes or use the root variables directly.

What’s Included in the CSS File?

The provided CSS file defines a scalable typography system using CSS variables. It includes:

  • Custom properties (--root-font-body, --root-scale-ratio, etc.)
  • Responsive typography adjustments for desktop, tablet, and mobile breakpoints
  • Root-level typography styles for h1 to h6 and p
  • Class-based typography styles for utility-based usage
  • Body text variations: .body-sm, .body-md, .body-lg

Available Classes

You can use the following classes to apply predefined typography styles:

        .h1 { font-size: var(--class-h1); }
        .h2 { font-size: var(--class-h2); }
        .h3 { font-size: var(--class-h3); }
        .h4 { font-size: var(--class-h4); }
        .h5 { font-size: var(--class-h5); }
        .h6 { font-size: var(--class-h6); }
        .body-sm { font-size: calc(var(--class-font-body) / var(--class-scale-ratio)); }
        .body-md { font-size: var(--class-font-body); }
        .body-lg { font-size: calc(var(--class-font-body) * var(--class-scale-ratio)); }

Customizing Root Font Size & Scale

To adjust the typography across different breakpoints, modify the CSS variables in the root. This ensures responsive scaling. 

                    :root {
                        /* Root-Level Base (Default for h1–h6, p) */
                        --root-font-body: 16px;
                        --root-scale-ratio: 1.25;
                    
                        /* Class-Level Base (For .h1, .h2, .body-md, etc.) */
                        --class-font-body: 18px;
                        --class-scale-ratio: 1.333;}

                    /* Tablet (Medium Screens) */
                    @media (max-width: 1024px) {
                        :root {
                            --root-font-body: 14px;
                            --root-scale-ratio: 1.2;
                            --class-font-body: 16px;
                            --class-scale-ratio: 1.3;
                        }
                    }
                    
                    /* Mobile (Small Screens) */
                    @media (max-width: 768px) {
                        :root {
                            --root-font-body: 12px;
                            --root-scale-ratio: 1.15;
                            --class-font-body: 14px;
                            --class-scale-ratio: 1.25;
                        }
                    }

--class-font-body: Base font size for body text.
--class-scale-ratio: Scale ratio for heading sizes and larger text.
Modify these values in the :root selector to achieve the desired typography adjustments.
